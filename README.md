# Slackbot - To Do

## Commands

### 1. /addtodo [task name]
- add a new task to TODO list
- duplicate task names allowed
### 2. /listtodos
- list all the task in todo list
### 3. /listdone
- list all the tasks marked as done
### 4. /marktodo [task name]
- mark a task as done
### 4. /cleartodolist
- delete all the tasks(Marked/unmarked) from the list
